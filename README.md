## About
ある日、会社の先輩からカメラをいただいたが全然撮りに行ったりせず一向にうまくならないので、これは強制力が必要であると察した。

撮影テーマ・撮影期限を決めることで強制的に自分のケツを叩き、撮影に向かわせるためのアプリ。

というのは手前上で、ただ作りたかっただけです🍞

## Libraries
